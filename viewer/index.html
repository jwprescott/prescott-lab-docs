How To Read This Viewer

Each slice image combines three panels: CT head, saliency overlay, and class probability bars. In the saliency map, warmer colors (yellow/red) indicate regions the model relied on more strongly, while cooler colors (blue/dark) indicate weaker contribution. In the bar chart, each bar is the predicted probability for one class. In general, a class probability greater than 0.5 suggests the finding is present, and the red bar marks the highest predicted value. Depending on file size and connection speed, slice images may take a few seconds to load.
